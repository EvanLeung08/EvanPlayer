快速入门

1 打开视频

点击工具栏左侧 📹 按钮选择视频文件。支持 MP4、MKV、AVI、MOV、WEBM、FLV、WMV、M4V 等几乎所有格式。

💡 macOS 用户：也可以直接将视频文件拖拽到播放界面打开。

2 加载字幕

EvanPlayer 会自动检测视频同目录下的同名字幕文件（.srt / .vtt / .ass / .ssa）并自动加载。MP4/MKV 中的内嵌字幕也会自动提取。

如需手动加载字幕：点击工具栏 📝 字幕按钮，选择字幕文件即可。

💡 双语字幕：点击工具栏 🌐 翻译字幕按钮，可以额外加载一份翻译字幕（如中文 .srt）。点击句子时自动匹配双语对照，无需消耗 AI 额度。

3 点击查词

视频播放时，点击字幕中的任意单词即可弹出释义窗口，显示离线词典释义（音标、英英释义、中文翻译、考试标签）。

长按字幕文本可获取整句翻译。翻译自动按优先级匹配：双语字幕 → Google 翻译 → 离线词典逐词释义 → AI 深度分析。

4 AI 翻译与分析

在释义窗口中点击「AI 智能翻译」按钮，可获取 AI 生成的英英释义、中英对照翻译和例句。需先在设置中配置 AI 服务。

EvanPlayer 支持两种 AI 模式：

  • 托管模式（推荐）：使用 EvanPlayer 服务器中转，无需自行申请 API Key，按额度使用。
  • 自定义模式：填入你自己的 AI API Key（如 DeepSeek 等），直连 AI 服务。也可使用 Ollama 本地 AI。

5 听写练习

在右侧面板切换到「听写」标签页。视频将自动循环当前字幕句，在输入框中键入听到的内容，然后点击「检查」查看逐词对比结果：

  • ■ 绿色 = 正确
  • ■ 红色 = 错误 / 拼写错误
  • ■ 橙色 = 遗漏的单词
  • ■ 灰色 = 多余的单词

支持渐进提示：第一次提示显示首字母，第二次显示中文释义，逐步帮你回忆。错词可一键加入生词本。

6 生词本与间隔复习

查词后点击「保存到生词本」，词汇将记录来源视频和字幕上下文。在「词汇」标签页中可查看所有保存的单词。

点击「复习」进入拼写测验模式（每组 20 词），基于 SM-2 间隔重复算法安排复习频率。四级掌握度：

  • 🆕 新词 — 刚加入，尚未复习
  • 📗 学习中 — 开始学习，需要巩固
  • 📘 复习中 — 已掌握基本含义，定期复习
  • ✅ 已掌握 — 可以长期记忆

生词本支持导出为 Excel (.xlsx) 或 CSV 格式，方便在其他工具中使用。

操作指南

播放控制

⏯ 播放 / 暂停
点击播放按钮，移动端双击视频区域
⏩ 快进 / 快退
拖动进度条，或点击上一句/下一句按钮
🔄 变速播放
点击速度按钮，选择 0.5x – 2.0x
🔊 音量调节
拖动音量滑块，或点击静音按钮
🔁 A-B 循环
点击循环按钮锁定当前字幕句反复精听
🖥 全屏模式
点击全屏按钮进入沉浸式播放

字幕功能

👁 显示模式
三种模式切换：显示 / 隐藏 / 悬停显示
🎭 字幕遮罩
CC 按钮开启遮罩，拖拽调整高度 5%–50%
🔍 字幕搜索
字幕列表顶部输入关键词快速定位
📌 字幕跳转
点击字幕列表任意一句跳转到该时间点
💡 听写小技巧：进入听写模式时 A-B 循环会自动设置到当前字幕句，退出时自动清除。EvanPlayer 每 5 秒自动保存播放位置，下次打开同一视频自动恢复。

常见问题
视频无法播放？

EvanPlayer 使用 libmpv 引擎，支持几乎所有常见视频格式。如果遇到无法播放：

  • 确认文件未损坏 — 尝试用其他播放器打开
  • 确认格式受支持（MP4、MKV、AVI、MOV、WEBM、FLV、WMV 等）
  • Android 用户：如果频繁闪退，尝试重启 App
字幕显示乱码？

EvanPlayer 已内置 UTF-8 和 UTF-16 自动检测。如果仍然乱码，可能是字幕使用了其他编码（如 GBK）。请用文本编辑器将字幕文件另存为 UTF-8 编码后重新加载。

macOS 版本提示字幕文件无法访问？

Mac App Store 版运行在沙箱中，无法自动读取视频同目录的其他文件。当自动加载字幕失败时会提示授权。

解决方法：点击提示中的「授权访问」按钮，选择视频所在文件夹并授权即可。之后同一文件夹下的字幕文件将可正常加载。

如何配置自定义 AI（自定义模式）？

设置 → AI 配置 → 切换到「自定义」模式：

  1. 前往你选择的 AI 服务商官网（如 DeepSeek）注册并创建 API Key
  2. 在 EvanPlayer 设置中填入 API Key
  3. Base URL 填写对应服务商地址（如 https://api.deepseek.com/v1）
  4. 模型按需选择（如 deepseek-chat）
  5. 点击「测试连接」验证配置

支持所有兼容接口的 AI 服务，只需填写对应的 Base URL、API Key 和模型名称即可。

如何配置 Ollama（本地 AI，完全免费）？

Ollama 允许在本地运行 AI 模型，完全免费、无需联网、数据不离开你的电脑。

  1. 前往 ollama.com 下载安装 Ollama
  2. 打开终端运行：ollama pull qwen2.5:3b
  3. 在 EvanPlayer 设置中选择自定义模式 → Ollama
  4. 地址默认 http://localhost:11434，模型填 qwen2.5:3b
💡 模型推荐：qwen2.5:3b 适合大多数电脑。Apple M 芯片或独显用户可以尝试 qwen2.5:7b 获取更高质量翻译。
离线词典如何安装？

进入设置页面，点击「下载离线词典」按钮即可自动下载（约 50MB）。词典基于 ECDICT 开源项目，含约 77 万词条。

下载源优先使用阿里云 CDN（国内高速），自动回退到备用源。安装完成后无需联网即可查词。

我的数据存储在哪里？

所有数据存储在本地设备上：

  • macOS：~/Library/Containers/com.evan.evanplayer/Data/.evanplayer/
  • iOS：App 沙箱内 Documents 目录（通过「文件」App 可查看）
  • Android：App 私有存储目录

包括 SQLite 数据库（设置、生词本、听写记录、播放历史）和离线词典。卸载 App 自动删除所有数据。

如何导出生词本？

在词汇标签页中点击「导出」按钮，选择 Excel (.xlsx) 或 CSV 格式。导出内容包括：单词、英文释义、中文翻译、例句、来源视频、添加日期。可通过系统分享功能发送到其他应用。

如何使用字幕遮罩功能？

有些视频的字幕是硬编码在画面中的，字幕遮罩可以在视频底部显示黑色遮罩条来遮挡：

  1. 点击控制栏的 CC 遮罩按钮开启遮罩
  2. 遮罩顶部有拖拽手柄，上下拖动调整高度
  3. 高度可在 5%–50% 之间调整，匹配不同视频的字幕位置
  4. 再次点击 CC 按钮关闭遮罩
