快速入门

1. 打开视频

菜单 → 文件 → 打开视频（⌘O），或直接将视频文件拖拽到播放界面。支持 MP4、MKV、AVI、MOV、WEBM 等格式。

2. 加载字幕

EvanPlayer 会自动检测视频同目录下的同名字幕文件（.srt / .vtt / .ass）。也会自动提取 MP4/MKV 中的内嵌字幕。如需手动加载：菜单 → 字幕 → 打开字幕文件。

3. 点击查词

视频播放时，点击字幕中的任意单词即可弹出释义窗口。按住 Shift 点击可以选取多个词/短语进行整句翻译。

4. AI 翻译

在弹出的释义窗口中点击「AI 智能翻译」按钮，可获取 AI 生成的英英释义、中英对照翻译和例句。需先在设置中配置 OpenAI API Key 或本地 Ollama。

5. 听写练习

在右侧面板切换到「听写」标签页。视频将自动循环当前字幕句，你可以在输入框中键入听到的内容，然后点击「检查」查看逐词对比结果。

6. 生词本

查词后点击「保存到生词本」，词汇将记录来源视频和字幕上下文。在「词汇」标签页中可查看所有保存的单词，并进行 SM-2 间隔重复复习。

常见问题

视频无法播放 / 提示格式不支持？

EvanPlayer 使用多引擎架构：

  • MP4/MOV 等常见格式由 JavaFX 引擎直接播放
  • MKV/AVI 等格式需要 VLC（请前往 videolan.org 下载安装）或 FFmpeg

安装 VLC 后重新打开视频即可。

字幕显示不正确 / 乱码？

EvanPlayer 已内置自动字符编码检测。如果仍然乱码，可能是字幕文件使用了非 UTF-8 编码。请用文本编辑器将字幕文件另存为 UTF-8 编码后重新加载。

字幕和视频不同步？

使用快捷键调整字幕时间轴偏移：

  • ⌘→ 字幕延后 100ms
  • ⌘← 字幕提前 100ms

多次按下直到字幕与语音同步。

如何配置 OpenAI？

菜单 → 设置 → 选择 OpenAI → 填入你的 API Key。

API Key 获取方式：前往 platform.openai.com 注册账号并创建 API Key。推荐使用 GPT-4o-mini 模型（性价比最高）。

你也可以使用任何 OpenAI 兼容的 API 服务（如 DeepSeek、Groq 等），只需修改 Base URL。

如何配置 Ollama（本地 AI）？

Ollama 允许你在本地运行 AI 模型，完全免费、无需联网、数据不离开你的电脑。

  • 前往 ollama.com 下载安装 Ollama
  • 打开终端运行：ollama pull qwen2.5:3b
  • 在 EvanPlayer 设置中选择 Ollama，地址默认 http://localhost:11434
离线词典如何安装？

首次查词时如果未检测到离线词典，会提示下载。点击下载按钮即可自动安装（约 180 MB 下载，解压后约 770 MB）。

词典基于 ECDICT 开源项目，包含约 77 万英语词条。

我的数据存储在哪里？

所有数据存储在你的 Mac 本地：

  • App Store 版本：~/Library/Containers/com.evan.evanplayer/Data/.evanplayer/
  • DMG 版本：~/.evanplayer/

包括：SQLite 数据库（设置、生词本、听写记录、播放历史）和离线词典。

如何导出生词本？

在词汇标签页中点击「导出」按钮，选择保存位置即可导出为 .xlsx (Excel) 格式。导出内容包括：单词、英文释义、中文翻译、来源视频、字幕上下文、添加日期。

键盘快捷键

快捷键 功能
Space 播放 / 暂停
← → 快退 / 快进 5 秒
↑ ↓ 音量 ±5%
S 切换字幕模式（显示 → 隐藏 → 悬停）
I 沉浸模式
F11 全屏
[ ] 设置 A-B 循环起点 / 终点
\ 清除 A-B 循环
⌘← ⌘→ 字幕偏移 ±100ms

系统要求

  • macOS 13 (Ventura) 或更高版本
  • Apple Silicon (M1/M2/M3/M4) 或 Intel Mac
  • 至少 200 MB 可用磁盘空间（安装离线词典需额外 ~800 MB）
  • AI 翻译功能需要网络连接（使用 OpenAI）或安装 Ollama（本地 AI）
