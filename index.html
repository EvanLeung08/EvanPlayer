核心功能

🎬

全格式视频播放

基于 libmpv 引擎，支持 MP4、MKV、AVI、MOV、WEBM、FLV、WMV、M4V 等几乎所有视频格式。变速播放（0.5x–2.0x）、A-B 循环、自动记忆播放位置、沉浸式全屏。

📝

智能字幕系统

支持 SRT / VTT / ASS / SSA 外挂字幕和 MP4/MKV 内嵌字幕自动提取。三种显示模式：显示、隐藏、悬停显示。支持加载双语翻译字幕、字幕遮罩（隐藏硬字幕）。

🤖

AI 多引擎翻译

点击字幕任意单词即刻查词。整句翻译自动匹配最佳来源：双语字幕 → Google 翻译 → 离线词典逐词释义 → AI 深度分析。支持 OpenAI / Ollama 本地 AI / 兼容 API（DeepSeek 等）。

📖

77 万词条离线词典

内置 ECDICT 离线词典，音标、英英释义、中文翻译、考试标签（四六级 / 考研 / 雅思 / 托福 / GRE）一应俱全。无需联网，随时查词。

✍️

听写练习 + AI 批改

逐句听写同步视频播放，自动循环当前字幕。逐词对比批改：正确（绿）、错误（红）、遗漏（橙）、多余（灰）。渐进提示系统，错词一键加入生词本。

🧠

间隔重复生词本

SM-2 算法科学安排复习。四级掌握度：新词 → 学习中 → 复习中 → 已掌握。TTS 朗读拼写测验（每组 20 词）。支持导出 Excel / CSV。

🌐

双语字幕对照

支持加载翻译字幕（如中文字幕），与英文字幕同步对照显示。点击整句时自动匹配翻译字幕，无需消耗 AI 额度即可获取翻译。

🎭

字幕遮罩

一键开启底部黑色遮罩条，遮挡视频中的硬编码字幕。可拖拽调整遮罩高度（5%–50%），精准匹配不同视频的字幕位置。

📱

三端同步体验

支持 macOS、iOS（iPhone / iPad）和 Android 三个平台。统一的暗色毛玻璃主题，中英双语界面一键切换。
