隐私政策

EvanPlayer - AI 智能英语学习视频播放器

首页 帮助 隐私政策

最后更新日期：2026 年 3 月 21 日

EvanPlayer（以下简称"本应用"）非常重视你的隐私。本隐私政策说明了本应用如何收集、使用和保护你的信息。

1. 信息收集

本应用不收集任何个人身份信息。

本应用不使用任何第三方分析、广告、追踪或统计 SDK。不存在用户账户系统，不要求注册或登录。

2. 本地数据存储

本应用在你的设备上本地存储以下数据，仅用于提供应用功能：

  • 应用设置：界面语言、字幕外观、播放偏好等配置项
  • 生词本：你保存的单词、释义、来源视频信息
  • 听写记录：听写练习的准确率和历史
  • 播放历史：视频的播放进度，用于断点续播
  • 离线词典：英语词典数据库（用户主动下载）

所有数据仅存储在你的 Mac 上，不会上传到任何服务器。

3. 网络通信

本应用在以下场景中会进行网络通信：

  • AI 翻译服务：当你使用 AI 查词/翻译功能时，应用将你查询的单词或句子发送到你自行配置的 AI 服务（OpenAI API 或本地 Ollama 服务器）。本应用不经手、不转发、不存储任何 API 通信内容。
  • 离线词典下载：首次使用时可选下载 ECDICT 离线词典，从 GitHub 公开仓库下载。

本应用不会在后台发起任何网络请求。所有网络通信均由用户主动触发。

4. 第三方服务

本应用支持接入以下第三方 AI 服务（由用户自行选择和配置）：

  • OpenAI API：使用你自己的 API Key 直连 OpenAI 服务器。你的 API Key 仅存储在本地设备上。请参阅 OpenAI 隐私政策。
  • Ollama：运行在你本地电脑上的 AI 服务，所有数据不离开你的设备。

本应用本身不运营任何云端服务。

5. 敏感权限

本应用不请求以下敏感权限：

  • 麦克风 / 摄像头
  • 通讯录 / 日历
  • 位置信息
  • 照片图库
  • 健康数据

本应用仅请求以下权限：

  • 文件访问：通过系统文件选择器打开你选择的视频和字幕文件（只读），以及保存导出的词汇表（写入）
  • 网络访问：用于 AI API 调用和离线词典下载

6. 儿童隐私

本应用不针对 13 岁以下的儿童，不故意收集儿童的任何信息。

7. 数据删除

卸载本应用将自动删除所有本地存储的数据。你也可以在应用运行期间手动删除生词本、听写记录等数据。

8. 隐私政策变更

如本隐私政策有更新，我们将在本页面发布新版本并更新"最后更新日期"。建议你定期查看本页面。
